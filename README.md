# OpenCart Automation Framework 🛒✨

A robust and scalable End-to-End (E2E) automation testing framework designed for the **OpenCart** e-commerce platform. Built from scratch using **TypeScript** to ensure type safety, speed, and reliability.

## 🚀 Features
* **Full E2E Coverage:** Automates critical user journeys (Registration, Login, Shopping Cart, Checkout).
* **Page Object Model (POM):** Clean, maintainable, and reusable code structure.
* **TypeScript Powered:** Enhances developer experience with strong typing and autocompletion.
* **Robust Reporting:** Integrated with detailed test reports (HTML/Allure).

## 🛠️ Tech Stack
* **Language:** TypeScript
* **Testing Framework:** [Playwright]
* **Design Pattern:** Page Object Model (POM)
